Warcraft Greetings

Great tae meet ya.

This is a /Command Bot built for Slack that displays random greetings from NPCs from the World of Warcraft game.

It is activated with /wow greeting.
It is further customizable with /wow farewell and can even return quotes from a specific race /wow greeting dwarf.
Use /wow races if you're unfamiliar with the World of Warcraft characters.

Learn more about Slack here, and more about World of Warcraft here.

Off with ye.
